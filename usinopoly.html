Bienvenue, industriel.

Partez avec 100 €. Achetez des usines (50 € la première, le prix grimpe à chaque achat) pour générer un revenu passif. Avec vos bénéfices, construisez des fusées pour bombarder les usines de vos adversaires et freiner leur empire. Le dernier baron industriel debout — ou le plus riche au temps imparti — remporte la partie.

Connexion directe entre navigateurs (sans serveur intermédiaire), en 3 étapes simples :

  1. Cliquez sur "Générer un code d'invitation" et envoyez le code A à votre ami (message, mail, peu importe).
  2. Votre ami colle ce code dans "Rejoindre une partie" et vous renvoie le code B généré.
  3. Collez ce code B ci-dessous et cliquez sur "Valider la connexion".

    Seul l'hôte peut lancer la partie. Vous pouvez générer une nouvelle invitation pour chaque joueur supplémentaire.
